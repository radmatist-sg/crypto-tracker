Cloud Sync — phone & laptop (Supabase)

Keep the same trades on every device. Set this up once per device — paste the three values below and hit Connect. Use the same sync code on your phone and laptop to link them.

First-time setup — create your free Supabase backend (≈5 min)
  1. Go to supabase.com → sign up (free) → New project. Pick any name & password, wait ~2 min for it to build.
  2. Open the SQL Editor (left sidebar), paste the snippet below, and click Run:
create table if not exists tracker (
  id text primary key,
  data jsonb,
  updated_at timestamptz default now()
);
alter table tracker enable row level security;
create policy "sync access" on tracker
  for all to anon using (true) with check (true);
  1. Go to Project Settings → API. Copy the Project URL and the anon public key into the boxes above.
  2. Click Generate for a sync code, then Connect.
  3. On your phone, open the same site, paste the same three values (same sync code!), and Connect. Done — they now share data.

Your data sits in your own private Supabase project. The anon key only lives on your devices, never in the public page.

Solana — Live Prices (DexScreener)

Prices for open Solana trades are fetched automatically from DexScreener — no API key needed. Covers virtually every token including new meme coins.

Solana — Helius API Key (required)

The public Solana RPC blocks browser requests. Helius is free and takes 30 seconds to set up.

  1. Go to helius.dev and sign up (free)
  2. Copy your API key from the dashboard
  3. Paste it below and hit Save
